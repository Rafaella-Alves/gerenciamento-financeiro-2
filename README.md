# Gerenciamento Financeiro 2
Este projeto foi desenvolvido durante o 2º período da faculdade, como parte dos meus estudos iniciais em **Java**. 
O foco principal foi praticar a lógica de programação, a manipulação de arquivos de texto e os primeiros passos na organização de código.

## 🚀O que o sistema faz
* Cadastro de Usuários: Coleta e organiza informações como nome, contato e endereço.
* Armazenamento em Arquivo: Salva as informações em arquivos .txt, permitindo que os dados não sejam perdidos ao fechar o programa.
* Divisão de Pastas: Organização dos arquivos Java separando as responsabilidades (Dados, Endereço, Contato).
  
## 📚Aprendizados Técnicos
* Introdução à POO: Uso de classes para representar entidades do mundo real (como Usuario e Endereco).
* Persistência de Dados: Leitura e escrita de arquivos (File I/O) para simular um banco de dados simples.
* Estruturas de Controle: Uso de loops e condicionais para gerenciar o fluxo do sistema.
  
## 🛠️Tecnologias Utilizadas
* Linguagem: **Java**
* IDE: **IntelliJ IDEA**
